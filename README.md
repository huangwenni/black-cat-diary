# black-cat-diary
黑猫日记，一个Web日记应用。Nodejs+Express+Vue+Es6+Mongodb。


作为一个练习，完成了简单的功能： 

1、登陆、注册、退出。

2、发布日记 

3、删除日记 

4、隐藏日记 

5、发布日记到树洞 

6、喜爱及留言 

7、更改用户信息
